# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [8.0.0] - 2026-04-18
### Added
- Industry-standard open-source readiness assets.

## [6.0.0] - 2026-04-18
### Added
- Real-time incremental indexing using Chokidar.
- Performance optimizations for background file watching.

## [5.0.0] - 2026-04-18
### Added
- Universal filesystem abstraction layer for parity between CLI and Extension.
- Multimodal document support for PDF and XLSX files.

## [4.0.0] - 2026-04-18
### Added
- BM25 relevance engine for advanced semantic context reranking.

## [3.0.0] - 2026-04-18
### Added
- Glossy AI chat UI for the VS Code Sidebar.
- Efficiency dashboard instrumentation for indexing telemetry.

## [2.0.0] - 2026-04-18
### Added
- Industry-grade symbol extraction & indexing core.

## [1.0.0] - 2026-04-18
### Added
- Initial extension scaffold.
- TOON (Token Oriented Object Notation) specification.
