# Contributing to Tooning

First off, thank you for considering contributing to Tooning! It's people like you who make it a great tool for the community.

## How Can I Contribute?

### Reporting Bugs
- Use the **GitHub Issues** tab.
- Explain the behavior you expected and what actually happened.
- Provide a minimal reproduction project if possible.

### Suggesting Enhancements
- Open a GitHub Issue and label it as an `enhancement`.
- Describe the features and why they would be useful.

### Pull Requests
- Fork the repo and create your branch from `master`.
- If you've added code that should be tested, add tests.
- Ensure the test suite passes (`npm test`).
- Make sure your code lints (`npm run lint`).

## Development Setup

1. **Clone the repo**: `git clone https://github.com/your-username/tooning.git`
2. **Install dependencies**: `npm install`
3. **Run the extension**: Use `F5` in VS Code to launch the Extension Development Host.
4. **Run tests**: `npm test`
5. **Watch CLI**: `npm run watch` then `node dist/bin/toon.js ./`
